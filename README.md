# High torque training

A playground where I experiment and learn about high torque (or low cadence) training inspired by [this video/podcast](https://www.youtube.com/watch?v=tcar3G2v73I).

---

Go visit [high-torque.jelen.dk](https://high-torque.jelen.dk/) to see stuff.

## Development

### Deployment

Deployed as a static side on GitHub pages. See [.github/workflows/deploy.yml](.github/workflows/deploy.yml).

### Site analytics

No analytics out of the box for GitHub Pages, so I decided to use GoatCounter, a basic privacy-friendly web analytics.

[Analytics dashboard here](https://jelen.goatcounter.com/).

Note that ad-blockers blocks it. See: [https://jelen.goatcounter.com/help/faq#blocked](https://jelen.goatcounter.com/help/faq#blocked).

> By my estimate about a third of pageviews are missed due to adblockers; but this can vary greatly on the type of site and audience.
